# Prerequisites
- JDK 1.8
- Maven 3.6+ 

# Installation
```xml
<dependency>
    <groupId>com.wutsi</groupId>
    <artifactId>wutsi-payment</artifactId>
    <version>[LATEST VERSION]</version>
</dependency>
```

Package available [here](https://github.com/wutsi/wutsi-payment/packages)
